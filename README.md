Ruby运行环境：2.0.0+

所需Gem: descriptive_statistics

````
gem install descriptive_statistics
````

运行：

````
ruby calc_dist.rb
````

运行之后，输入点的经度、维度、高度
可以按照逗号分隔，也可以逐个输入。
